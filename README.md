# 住宅ローン計算機

モダンなデザインと高度な計算ロジックを備えた、高機能な住宅ローンシミュレーターです。

## ✨ 特徴

- **2つの返済方式に対応**: 元利均等返済と元金均等返済の切り替えが可能。
- **ボーナス払いシミュレーション**: ボーナス併用時の支払額と、月々のみの支払額をリアルタイムに比較。
- **詳細な返済予定表**: 完済までの全期間の明細を表示。1年ごとの集計表示にも対応。
- **印刷・PDF出力最適化**: A4縦1〜2枚に収まるよう、印刷専用のレイアウトを設計。
- **レスポンシブデザイン**: PC、タブレット、スマートフォンすべてで快適に操作可能。
- **入力補完**: 数値入力とレンジスライダーが連動し、直感的な操作を実現。

## 🚀 使い方

1. `index.html` をブラウザで開くだけで使用できます。
2. 借入額、期間、金利、ボーナス額を入力すると即座に結果が反映されます。
3. 必要に応じて「印刷 / PDF保存」ボタンから結果を出力してください。
